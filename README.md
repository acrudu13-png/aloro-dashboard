# Aloro Dashboard

Enterprise voice AI platform dashboard for managing assistants, call logs, and analytics.

## Features

- **Dashboard** - KPIs, call volume charts, outcomes widget
- **Call Insights** - Post-call variable analytics, escalation tracking
- **Assistants** - Multi-step wizard with tools, variables, LLM config
- **Call Logs** - Filterable call history with extracted variables

## Tech Stack

- HTML5 + Tailwind CSS (CDN)
- Vanilla JavaScript + jQuery
- No build step required

## Development

```bash
# Open locally
open index.html

# Or serve with any static server
npx serve .
```
